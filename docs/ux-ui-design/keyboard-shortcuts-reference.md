# Keyboard Shortcuts Reference

## Overview

Hardware Tool provides comprehensive keyboard shortcuts for efficient workflow. All shortcuts are customizable, and the system detects conflicts automatically.

## Customization

### Shortcut Editor

```rust
ShortcutConfig {
    // Access via Preferences > Keyboard Shortcuts
    
    // Search and filter
    search_by_action: true,
    search_by_shortcut: true,
    
    // Conflict detection
    detect_conflicts: true,
    warn_on_override: true,
    
    // Presets
    presets: vec![
        Preset::Default,
        Preset::KiCad,
        Preset::Altium,
        Preset::Eagle,
        Preset::Vim,
    ],
    
    // Export/Import
    export_format: "json",
}
```

Press `Ctrl+?` to display an overlay with all shortcuts for the current context.
